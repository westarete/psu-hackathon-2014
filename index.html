PSU Hackathon

To be done before the Hackathon

  1. Download the text editor, Sublime Text 2, which is available for both Mac and Windows; http://www.sublimetext.com/2 (32 vs. 64 bit Windows? -> see 'Stickiness 2.' below)
  2. Make sure you have Ruby installed

  3. MAC ONLY: most likely you already have ruby installed (since it ships standard with a Mac)

    1. to check which version of ruby you have installed, go to the command line and type: ruby -v
    2. you're good to go if you get back something that looks like this: ruby 2.0.0p247
  • WINDOWS ONLY: download ruby from: http://rubyinstaller.org/downloads/
    1. we'll be using ruby 1.9.3 or greater for this workshop
    2. install ruby 1.9.3-p545
    3. during the installation process, you'll be asked to:
      1. install Td/Tk support (answer: NO)
      2. add ruby executables to your PATH (answer: YES)
      3. associate .rb and .rbw files with this ruby installation (answer: YES)

And during the Hackathon

  1. Create a new file in Sublime Text2
  2. put this code in the file you just created:
require 'sinatra' get '/' do
  "Hello World!"
end
  1. save the file as 'myapp.rb'
  2. go to the command line
    • navigate to the directory where that file is
    • enter this on the command line; gem install sinatra (good on Mac or Windows)
      • could be ticklish
    • enter this on the command line ruby myapp.rb
  3. open your web browser
  4. paste this where the URL goes: http://localhost:4567
